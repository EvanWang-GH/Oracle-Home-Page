# Oracle-Home-Page
仿 Oracle 首页
[在线预览（可能需要翻墙）](https://evanwang-gh.github.io/Oracle-Home-Page/)  
共制作了 1 个主页面和 3 个子页面。
其中，主页面的布局和动画较为复杂，使用了 grid 布局，并结合了 jQuery 实现了多种动画效果。
在子页面使用了 bootstrap 框架。
